4 Security, Governance & Cost Management Microsoft Entra ID · Agent 365 · Cost Controls

🔐 Microsoft Entra Agent ID – assign unique service principal per agent, enforce conditional access.

⚙️ Microsoft Agent 365 – central control plane for agent discovery, access policies, compliance at scale.

💰 Agent Cost Controls – throttle tokens, limit conversation turns, set budgets per agent/department.

🏛️ Governance architecture – Agent 365 control plane
📌 Real-world governance scenario

IT Helpdesk agents: Deploy 50 Copilot agents per region. Use Agent 365 to enforce PII redaction, monthly token budget of $500 per agent, and Entra ID MFA for privileged topics. Cost anomaly detection alerts if agent consumes 200% of expected tokens.

5 Customize, Orchestrate & Experiment (Azure AI Foundry) RAG · Fine-tuning · Distillation · Agent Service

Retrieval Augmented Generation (RAG) + Fine-tuning & Distillation for domain-specialized agents. Azure AI Foundry Agent Service enables agent orchestration and debugging.

📚 RAG + Fine-tuning: Financial analyst agent

Ingest 10k SEC filings into Azure AI Search → RAG grounds responses. For complex sentiment, fine-tune a GPT-4o model on past earnings calls. Distill to a smaller model for cost efficiency. Use Azure AI Foundry to debug orchestration flow.

# Foundry evaluation snippet from azure.ai.resources import AIProject client = AIProject.from_connection("fin-rag-project") eval_results = client.evaluate(agent_id="analyst-01", dataset="qa_finance.json") print(f"Grounding score: {eval_results.groundedness}")

7 Cross-Platform Orchestration: Copilot Studio + Azure Foundry MCP contract · Invocation Patterns · Enterprise scenarios

Agent invocation patterns: Copilot Studio invokes Foundry agents for advanced reasoning; Foundry invokes declarative Copilot agents for conversational UX. MCP (Model Context Protocol) as common contract layer.

🏢 Enterprise Scenario: IT Helpdesk + Foundry reasoning

IT Helpdesk Copilot Studio agent (Teams chat) receives complex "debug log analysis" request → Invokes Azure AI Foundry reasoning agent that runs semantic kernel planner, queries logs, returns root cause. The Copilot agent formats response for end-user. Cost-efficient: reasoning only triggered for complex tasks.
