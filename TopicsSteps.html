Copilot Studio Beginner Tutorial

Simple Example to Explain Agent and Topics

This tutorial explains what an Agent is, what Topics are, and how to create a simple Copilot Studio agent step by step using a small practical example.

What is an Agent?

An Agent is like a smart digital assistant. It talks to users, understands their questions, and gives answers or performs actions.

Simple example: A “Training Help Agent” that explains AI, Python, RAG, MCP, and Copilot Studio topics.

What is a Topic?

A Topic is one small conversation flow inside the agent. Each topic handles one specific user need.

Simple example: “Explain RAG”, “Explain MCP”, “Give Interview Questions”, or “Troubleshoot Error”.

1. Very Simple Understanding

Copilot Studio TermMeaningSimple Example
Agent The full chatbot or AI assistant. Technical Training Assistant
Topic One conversation flow inside the agent. Explain Topic
Trigger Phrases Sentences that start a topic. “Explain RAG”, “What is MCP?”
Message Node Agent sends a message to the user. “Sure, I can explain that.”
Question Node Agent asks the user for information. “Which topic do you want to learn?”
Variable Stores user input for later use. Store topic name in varTopic
Official concept: In Copilot Studio, a topic defines how an agent conversation progresses. Trigger phrases help the agent understand when a topic should start.

2. Simple Real-Time Example

Use Case: Training Help Agent

We will create a simple agent that explains any topic entered by the learner.

User asks: Explain RAG → Topic starts: Explain Topic → Agent explains RAG → Agent asks next option
Goal: User enters a topic name. The agent explains it in a simple trainer-style format.

3. Main Agent Prompt

Paste this in the Agent Instructions area.

Copy this prompt
You are a simple corporate technical training assistant.

Your job is to explain technical topics in simple English.

When the user asks about any topic, explain it using this format:

1. Meaning
2. Why we use it
3. Simple example
4. Real-world use case
5. Step-by-step explanation
6. Quick summary

Rules:
- Explain like a classroom trainer.
- Keep the answer beginner-friendly.
- Use short paragraphs.
- Use bullet points when useful.
- If the user asks for interview questions, give questions and answers.
- If the user asks for a practical example, give a simple real-time example.
- If you are not sure, ask for more details.

4. Step-by-Step: Create the Agent

Open Copilot Studio

Go to Microsoft Copilot Studio and sign in with your Microsoft account.

Create a New Agent

Click Create and select New agent.

Enter Agent Name

Use this name:

Training Help Agent

Enter Short Description

Use this description:

This agent explains technical topics in a simple beginner-friendly way.

Add Instructions

Paste the main agent prompt from the previous section into the Instructions area.

Create the Agent

Click Create. After the agent is created, go to the Topics section.

5. Step-by-Step: Create First Topic

Create one simple topic called Explain Topic.

Go to Topics

Open your agent and click Topics.

Add New Topic

Click Add a topic.

Name the Topic

Explain Topic

Add Trigger Phrases

Trigger phrases tell the agent when this topic should start.

Explain topic
Explain RAG
Explain MCP
What is GenAI
Teach me Python
Tell me about Copilot Studio
I want to learn

Add a Message Node

This is the first message shown to the user.

Sure, I can explain technical topics in a simple way.

Add a Question Node

Ask the learner which topic they want to learn.

Which topic do you want me to explain?

Save the answer in a variable:

varTopic

Add Final Message Node

Use this message to explain the topic.

Here is a simple explanation of {varTopic}:

1. Meaning:
{varTopic} is an important concept used in technology.

2. Why we use it:
We use {varTopic} to solve real-world problems and make work easier.

3. Simple example:
Think of {varTopic} like a helper that supports a task step by step.

4. Real-world use case:
Companies can use {varTopic} for automation, support, training, data handling, or decision making.

5. Summary:
{varTopic} is useful because it helps people understand and solve problems more effectively.

Do you want a practical example or interview questions for this topic?

Save and Test

Save the topic and test with:

Explain RAG

6. Sample Conversation Output

This is how the agent conversation should look.

User: Explain RAG

Agent: Sure, I can explain technical topics in a simple way.

Agent: Which topic do you want me to explain?

User: RAG

Agent:
Here is a simple explanation of RAG:

1. Meaning:
RAG means Retrieval Augmented Generation.

2. Why we use it:
We use RAG to help AI answer from trusted documents instead of guessing.

3. Simple example:
If an employee asks about leave policy, RAG searches the HR document and gives the correct answer.

4. Real-world use case:
RAG can be used in HR chatbots, customer support bots, document Q&A systems, and training assistants.

5. Summary:
RAG helps AI give more accurate answers using your own data.

Do you want a practical example or interview questions for this topic?

8. Prompt to Create Topic Automatically

If Copilot Studio gives you an option to describe the topic, paste this prompt.

Auto-create topic prompt
Create a topic called "Explain Topic".

The topic should explain any technical topic entered by the user.

Flow:
1. Start when the user says phrases like "Explain", "What is", "Teach me", or "I want to learn".
2. Ask the user: "Which topic do you want me to explain?"
3. Store the answer in a variable called varTopic.
4. Explain varTopic using this format:
   - Meaning
   - Why we use it
   - Simple example
   - Real-world use case
   - Summary
5. End by asking whether the user wants a practical example or interview questions.

Tone:
Simple, friendly, classroom trainer style.
